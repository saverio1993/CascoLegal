# Carpeta de Documentos PDF Oficiales - CascoLegal

Coloca en esta carpeta todos los archivos PDF originales de las Gacetas Oficiales y resoluciones de la ATTT de Panamá que deseas cargar en el sistema.

### Archivos recomendados para colocar aquí:
1.  **Decreto_Ejecutivo_640_2006.pdf** (Reglamento de Tránsito Vehicular completo).
2.  **Decreto_Ejecutivo_19_2022.pdf** (Decreto de delivery y rotulación de placa).
3.  **Resolucion_OAL_904_2013.pdf** (Normativa de chaleco reflectivo con placa).
4.  **Resolucion_OAL_973_2013.pdf** (Reglamentación y multas por chalecos).

El script de procesamiento extraerá el texto de estos archivos, segmentará cada artículo de forma automática y los indexará en la base de datos de CascoLegal para que la IA (Gemini / MiniMax) pueda responder con citas exactas y enlaces de descarga directa.
